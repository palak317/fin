# üöÄ FINWISE AI | The Intelligent Financial Cockpit

**FinWise AI** is a next-generation financial dashboard that bridges the gap between static bank statements, real-time spending, and AI-driven investment strategy. It combines **Generative AI (Gemini)** with **Live Market Data (Yahoo Finance)** and **WhatsApp integration** to create a seamless financial ecosystem.


## üåü Key Features

### 1. üß† Neural Analyzer (Statement Decryption)
* **Problem:** Bank statements are messy PDFs that are hard to analyze manually.
* **Solution:** Upload any PDF bank statement. Our **Gemini AI** model extracts transactions, categorizes them (Food, Travel, Investment) automatically, and visualizes spending patterns in interactive 3D charts.

### 2. üîÆ Wealth Oracle (Real-Time Strategy)
* **Problem:** AI usually gives generic financial advice without knowing current market prices.
* **Solution:** A hybrid engine. Gemini generates the *strategy* (e.g., "Buy Tata Motors"), and Python fetches the **Live Real-Time Price** from the NSE via Yahoo Finance to calculate exactly how many shares you can afford based on your budget.

### 3. üì≤ Live Tracker (WhatsApp Bridge)
* **Problem:** Budgeting apps fail because opening them is a chore.
* **Solution:** A dedicated **Twilio Webhook**. Just text *"Spent 200 on Coffee"* on WhatsApp, and the dashboard updates instantly without you ever opening the website.

## üöÄ How to Run Locally

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/yourusername/finwise-render.git](https://github.com/yourusername/finwise-render.git)
    cd finwise-render
    ```

2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the Application**
    ```bash
    python app.py
    ```

4.  **Access the Dashboard**
    Open `http://127.0.0.1:5000` in your browser.

5.  **Authenticate**
    Enter your **Google Gemini API Key** in the secure modal (stored locally in your browser for privacy).

## üì± WhatsApp Bot Setup (Twilio)

To enable the Live Tracker feature:

1.  Create a free account on [Twilio](https://www.twilio.com/).
2.  Go to **Messaging > Try it out > Send a WhatsApp message**.
3.  In the **Sandbox Settings**, set the **"When a message comes in"** webhook to:
    `https://your-app-name.onrender.com/webhook`
4.  Scan the QR code on the dashboard or text the join code (e.g., `join paper-flamingo`) to the bot.
5.  Start tracking: "Spent 500 on Pizza".
